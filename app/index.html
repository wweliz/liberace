ABOUT

Hi there! My name is Whitney.

I'm a designer-developer based in Greenville, South Carolina.

A true form-follows-function girl, I develop clean, simple solutions that speak across boundaries. My favorite projects center around a strong brand identity, especially those with multicultural and multilingual aspects.

WORK

Mice On Main The Game

Mice On Main is a webapp that utilizes the HTML5 geolocation API to help visitors to Greenville find the mice sculptures hidden along Main Street in Greenville, SC.

Build Summary:

HTML, Sass, JavaScript, jQuery, Underscore, local camera access, Parse SDK, HTML5 geolocation API

This app is still in development, but you can check out my code so far on GitHub.

AwesomeDice

AwesomeDice is a webapp that simulates rolling dice at a casino table. Users input the number of dice rolled and the number of sides on each die, and a then show a representation of the dice on the table.

Build Summary:

HTML, Sass, JavaScript

View the project live or check out my code on GitHub.

English Theatre Berlin Playbill

English Theatre Berlin is the German capital's only producing and presenting organization expressly dedicated to English as the working language. I designed and illustrated a poster for the play Prophecy of a Nameless Eskimo, which ran from April to May 2009.

Chomarat Rebrand
